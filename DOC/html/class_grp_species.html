The operation of consulting one of the possible prey species.

Allows us to get an id of a species with which we can alimentate current member animal.

Precondition
The implicit parameter is a valid group of species.
Postcondition
The id of a prey unit, registered in the prey vector of current member species under the prey[seq] is returned if it exists. If the member species' prey[seq] does not exist, -1 is returned.
